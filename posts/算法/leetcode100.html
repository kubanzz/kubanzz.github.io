  • 算法：HashMap
  • 误区：
    • 误区一：排序 + 双指针，这个算法排序的时候下标发生了变化，得出的不是预期的结果
    • 误区二：先遍历一次存入哈希，再遍历检查元素（会获取到自身元素）
public int[] twoSum(int[] nums, int target) {
Map<Integer, Integer> map = new HashMap<>();

// 误区二
for(int i = 0;i < nums.length;i ++){
   if(map.containsKey(target - nums[i])){
       return new int[]{i, map.get(target - nums[i])};
   }else {
       map.put(nums[i], i);
   }
}

return new int[]{-1, -1};
}

public int lengthOfLongestSubstring(String s) {

    int left = 0;
    int right = 0;
    Map<Character, Integer> windows = new HashMap<>();
    int maxVal = 0;

    while(right < s.length()){
        char c = s.charAt(right);
        windows.put(c, windows.getOrDefault(c, 0) + 1);
        right ++;

        while(windows.get(c) > 1){
            char d = s.charAt(left);
            windows.put(d, windows.get(d) - 1);
            left ++;
        }
        maxVal = Integer.max(maxVal, right - left);
    }

    return maxVal;
}

# 5. >最长回文子串

  • 解法：中心扩列法

  • 注意：substring需要的是左右的下标

  • 推荐解法：直接在findLongestPalindrome中判断和获取回文串

String res = "";
public String longestPalindrome(String s) {

    for(int i = 0;i < s.length();i ++){
        findLongestPalindrome(s, i, i + 1);
        findLongestPalindrome(s, i, i);
    }

    return res;
}

private void findLongestPalindrome(String s, int left, int right){

    while(left >= 0 && right < s.length()){
        if(s.charAt(left) == s.charAt(right)){
            if(right - left + 1 > res.length()){
                res = s.substring(left, right + 1);
            }
            left --;
            right ++;
        }else {
            return;
        }
    }

    return;
}

# 11. >盛最多水的容器

  • 双指针
    • 移动长板：面积一定变小，因为短板不变，长度变小
    • 移动短板：可能变大，长度虽然变小，但是下一个板可能比当前短板大
public int maxArea(int[] height) {

    int max = 0;
    int left = 0;
    int right = height.length - 1;

    while(left < right){
        max = Integer.max(max, Integer.min(height[left], height[right]) * (right - left));

        if(height[left] < height[right]){
            left ++;
        }else {
            right --;
        }
    }

    return max;
}

# 15. >三数之和

  • 三指针：排序，固定一指针，移动另外两个指针
  • 重点：去重，本质是跟上一个元素进行比较，比如相同则继续移动
public List<List<Integer>> threeSum(int[] nums) {

    Arrays.sort(nums);

    List<List<Integer>> ans = new ArrayList<>();
    for(int i = 0;i < nums.length;i ++){

        int left = i + 1;
        int right = nums.length - 1;

        // 去重
        if(i > 0 && nums[i] == nums[i - 1]){
            continue;
        }

        while(left < right){
            int sum = nums[left] + nums[right] + nums[i];
            if(sum == 0){
                ans.add(Arrays.asList(nums[left], nums[right], nums[i]));
                left ++;
                right --;

                // 去重
                while(left < right && nums[left] == nums[left - 1]) left ++;
                while(left < right && nums[right] == nums[right + 1]) right --;
            }else if(sum < 0){
                left ++;
            }else {
                right --;
            }
        }
    }

    return ans;
}

# 17. >电话号码的字母组合

  • 算法：回溯法（递归本质上就是一种深度遍历）

public List<String> letterCombinations(String digits) {

        return new ArrayList<>();
    }

    Map<Character, String> map = new HashMap<>();
    map.put('2', "abc");
    map.put('3', "def");
    map.put('4', "ghi");
    map.put('5', "jkl");
    map.put('6', "mno");
    map.put('7', "pqrs");

public void backtract(String digits, StringBuilder subAns, List<String> ans, Map<Character, String> map, int index){

    String nums = map.get(c);

        char c2 = nums.charAt(i);
        subAns.append(c2);
        backtract(digits, subAns, ans, map, index + 1);
        subAns.deleteCharAt(subAns.length() - 1);
    }

}

# 19. *删除链表的倒数第 N 个结点

  • 算法：快慢指针
  • 注意：添加dummyHead

删除倒数节点可以记为模版（删除倒数第n个节点）：

  • 快指针先移动n位（因为有dummyhead节点，不然的话移动n位自然就在目标节点处，这个很容易理解）
  • 快慢指针同时走，直到快指针走到尾部节点，不走null
public ListNode removeNthFromEnd(ListNode head, int n) {
    if(head == null || n <= 0){
        return head;
    }
    
    ListNode dummyHead = new ListNode();
    dummyHead.next = head;
    ListNode p1 = dummyHead;
    ListNode p2 = dummyHead;
	
    // 倒数第几个就走几步
    for(int i = 1;i <= n;i ++){
        p1 = p1.next;
    }
    
    // 在目标前一位停下 
    while(p1.next != null){
        p1 = p1.next;
        p2 = p2.next;
    }

    p2.next = p2.next.next;

    return dummyHead.next;
}

# 20. >有效的括号

  • 算法：利用栈
    • 遇到左符合则入栈
    • 遇到右符合则检查栈不为空且栈顶元素需为其左符号
  • 记录：❌
public boolean isValid(String s) {

    Deque<Character> stack = new ArrayDeque<>();
    Map<Character, Character> map = new HashMap<>();
    map.put('(', ')');
    map.put('[', ']');
    map.put('{', '}');

    for(char c : s.toCharArray()){
        if(map.containsKey(c)){
            stack.push(c);
        }else {
            if(stack.isEmpty() || map.get(stack.peek()) != c){
                return false;
            }
            stack.pop();
        }
    }

    return stack.isEmpty();
}

    }

}

# 22. 括号生成 (opens new window)

  • 算法：回溯法（全排列） - 暴力解法
public List<String> generateParenthesis(int n) {

    List<String> ans = new ArrayList<>();
    backtract(ans, new StringBuilder(), n, n);

    return ans;
}

public void backtract(List<String> ans, StringBuilder subAns, int left, int right){
    if(left == 0 && right == 0){
        String str = new String(subAns);
        if(valid(str)) ans.add(str);
        return;
    }

    if(left > 0){
        subAns.append("(");
        backtract(ans, subAns, left - 1, right);
        subAns.deleteCharAt(subAns.length() - 1);
    }

    if(right > 0){
        subAns.append(")");
        backtract(ans, subAns, left, right - 1);
        subAns.deleteCharAt(subAns.length() - 1);
    }

            }
        }
    }

public int maxArea(int[] heights) {

  Deque<Integer> stack = new ArrayDeque<>();
  int len = heights.length;
  int[] left = new int[len];
  int[] right = new int[len];
  Arrays.fill(right, len);

  for (int i = 0; i < heights.length; i++) {
    while (!stack.isEmpty() && heights[stack.peek()] >= heights[i]) {
      right[stack.peek()] = i;
      stack.pop();
    }
    left[i] = stack.isEmpty() ? -1 : stack.peek();
    stack.push(i);
  }

  int ans = 0;
  for (int i = 0; i < len; i++) {
    ans = Integer.max(ans, (right[i] - left[i] - 1) * heights[i]);
  }

  return ans;
}

    node.next = dummyHead.next;
    dummyHead.next = node;
    node.next.prev = node;

    int[] indeg = new int[numCourses];

        }

public int findKthLargest(int[] nums, int k) {

  int n = nums.length;
  int left = 0;
  int right = n - 1;
  int target = n - k;

  while(true){
    int pIndex = partition(nums, left, right);
    if(pIndex == target){
      return nums[pIndex];
    }else if (pIndex > target){
      right = pIndex - 1;
    }else {
      left = pIndex + 1;
    }

private int partition(int[] nums, int left, int right){

  int lt = left;
  int pivot = nums[left];

  for(int i = lt + 1;i <= right;i ++){
    if(nums[i] <= pivot){
      lt ++;
      swap(nums, i, lt);
    }
  }
  swap(nums, left, lt);

  return lt;
}

private void swap(int[] nums, int left, int right){
  int tmp = nums[left];
  nums[left] = nums[right];
  nums[right] = tmp;

  • 当前符号为1，则dp(i)(j) = Integer.min（dp-left，dp-right，dp-对角） + 1
public int maximalSquare(char[][] matrix) {

  int m = matrix.length;
  int n = matrix[0].length;
  int[][] dp = new int[m][n];
  int max = 0;

  for(int i = 0;i < m;i ++){
    if(matrix[i][0] == '1'){
      dp[i][0] = 1;
      max = 1;
    }
  }

  for(int i = 0;i < n;i ++){
    if(matrix[0][i] == '1'){
      dp[0][i] = 1;
      max = 1;
    }
  }

  for(int i = 1;i < m;i ++){
    for(int j = 1;j < n;j ++){
      if(matrix[i][j] == '1'){
        dp[i][j] = Integer.min(dp[i - 1][j], Integer.min(dp[i][j - 1], dp[i - 1][j - 1])) + 1;
        max = Integer.max(max, dp[i][j]);
      }
    }
  }

  return max * max;
}

public int[] productExceptSelf(int[] nums) {

  int n = nums.length;
  int[] ans = new int[n];
  ans[0] = 1;

  for(int i = 1;i < n;i ++){
    ans[i] = ans[i - 1] * nums[i - 1];
  }

  int right = 1;
  for(int i = n - 1;i >= 0;i --){
    ans[i] = ans[i] * right;
    right = right * nums[i];
  }

  return ans;
}

# *239. 滑动窗口最大值 (opens new window)

算法：滑动窗口 + 单调队列

思路：

  • 队尾（右指针移动更新）：单调栈存放单调递增值（如果队尾的元素小于当前元素，则队尾出队）

  • 队头（左指针移动更新）：保证队列中的元素下标都在窗口内

public int[] maxSlidingWindow(int[] nums, int k) {

  Deque<Integer> queue = new ArrayDeque<>();
  int left = 0;
  int right = 0;
  int n = nums.length;
  List<Integer> ans = new ArrayList<>();

  while(right < n){
    int c = nums[right];
    while(!queue.isEmpty() && nums[queue.peekLast()] < c){
      queue.pollLast();
    }
    queue.offer(right);
    right ++;

    while(right - left >= k){
      ans.add(nums[queue.peek()]);

      int d = nums[left];
      left ++;

      // 左指针移动一位，则判断队头是否已经不在窗口内
      if(!queue.isEmpty() && queue.peekFirst() < left){
        queue.pollFirst();
      }
    }
  }

  return ans.stream().mapToInt(i -> i).toArray();
}

public List<String> removeInvalidParentheses(String s) {
  int lremove= 0;
  int rremove = 0;

  for(char c : s.toCharArray()){
    if(c == '('){
      lremove ++;
    }else if(c == ')'){
      if(lremove > 0){
        lremove --;
      }else {
        rremove ++;
      }
    }
  }

  List<String> ans = new ArrayList<>();
  backtract(ans, s, lremove, rremove, 0);

  return ans;
}


public void backtract(List<String> ans, String s, int lremove, int rremove, int index){
  if(rremove == 0 && lremove == 0){
    if(valid(s)){
      ans.add(s);
    }
    return;
  }

  for(int i = index;i < s.length();i ++){
    char c = s.charAt(i);

    if(i > 0 && c == s.charAt(i - 1)){
      continue;
    }

    if(c == '(' && lremove > 0){
      backtract(ans, s.substring(0, i) + s.substring(i + 1), lremove - 1, rremove, i);
    }else if(c == ')' && rremove > 0){
      backtract(ans, s.substring(0, i) + s.substring(i + 1), lremove, rremove - 1, i);
    }
  }
}

private boolean valid(String s){
  int cnt = 0;
  for(char c : s.toCharArray()){
    if(c == '('){
      cnt ++;
    }else if(c == ')'){
      if(cnt > 0){
        cnt --;
      }else {
        return false;
      }
    }
  }

  return cnt == 0;
}

# 309. 最佳买卖股票时机含冷冻期 (opens new window)

算法：动态规划

思路：

  • dp(i)(j)：表示某天过后为某个状态的最大收益
    • 买入：max(昨天买入，今天买入)
    • 卖出：max(昨天买出，昨天冷冻期)
    • 冷冻期：今天卖出
public int maxProfit(int[] prices) {

  int n = prices.length;
  int[][] dp = new int[n][3];

  dp[0][0] = 0;
  dp[0][1] = -prices[0];
  dp[0][2] = 0;

  int max = 0;
  for(int i = 1;i < n;i ++){
    // 今天没有执行操作，卖出状态可能来自上个卖出状态，也可能由冷冻期转化而来
    dp[i][0] = Integer.max(dp[i - 1][0], dp[i - 1][2]);
    // 今天执行了买入操作，或者上个状态就是买入状态
    dp[i][1] = Integer.max(dp[i - 1][1], dp[i - 1][0] - prices[i]);
    // 今天执行了卖出操作
    dp[i][2] = dp[i - 1][1] + prices[i];

    max = Integer.max(max, Integer.max(dp[i][0], dp[i][2]));
  }

  return max;
}

# **416. 分割等和子集 (opens new window)

易忘题/重点题

算法：动态规划

注意：需要防止重复选择问题（关键），这道题跟零钱兑换问题最关键的不同是物品只能被拿一次，所以需要两个状态去维护，即dp[][]，而零钱兑换零钱是可以无限取的，所以是`dp[]``。

思路：转化为0-1背包问题，并且是只能取一次的0-1背包

  • dp[i][j]：表示前i个物品，有j的容量，能否放满。
    • 选择当前物品：则dp[i][j] = dp[i - 1][j - nums[i]]
    • 不选择当前物品：则dp[i][j] = dp[i-1][j]
public boolean canPartition(int[] nums) {

  int sum = Arrays.stream(nums).sum();
  if(sum % 2 != 0){
    return false;
  }
	
  sum = sum / 2;
  int n = nums.length;
  boolean[][] dp = new boolean[n][sum + 1];
	
  // 初始化第一行
  for(int i = 0;i < n;i ++){
    dp[i][0] = true;
  }
	
  // 初始化第一列
  if(nums[0] <= sum) dp[0][nums[0]] = true;
	
  // 遍历第一个状态
  for(int i = 1;i < n;i ++){
    // 遍历第二个状态
    for(int j = 1;j <= sum;j ++){
      dp[i][j] = dp[i - 1][j];
      if(j - nums[i] >= 0) dp[i][j] |= dp[i - 1][j - nums[i]]; 
    }
  }

  return dp[n - 1][sum];
}

public void backtract(TreeNode root, int targetSum, Long currentSum){
  if(root == null){
    return;
  }

  currentSum += root.val;
  count += prefix.getOrDefault(currentSum - targetSum, 0);
  prefix.put(currentSum, prefix.getOrDefault(currentSum, 0) + 1);

  backtract(root.left, targetSum, currentSum);
  backtract(root.right, targetSum, currentSum);

  prefix.put(currentSum, prefix.get(currentSum) - 1);
  currentSum -= root.val;
}

# 438. 找到字符串中所有字母异位词 (opens new window)

算法：滑动窗口（框架）

思路：

  • 一个当前窗口+一个需要的字符的窗口
  • valid：当前已经满足要求的字符数量，与need的大小相同时即窗口中所有字符都满足
  • 左右指针则根据题目进行移动
public List<Integer> findAnagrams(String s, String p) {

  Map<Character, Integer> windows = new HashMap<>();
  Map<Character, Integer> need = new HashMap<>();
  List<Integer> ans = new ArrayList<>();
  int valid = 0;
  int left = 0;
  int right = 0;

  for(char c : p.toCharArray()){
    need.put(c, need.getOrDefault(c, 0) + 1);
  }

  while(right < s.length()){

    char c = s.charAt(right);
    if(need.containsKey(c)){
      windows.put(c, windows.getOrDefault(c, 0) + 1);
      if(windows.get(c).equals(need.get(c))){
        valid ++;
      }
    }
    right ++;

    while(right - left >= p.length()){
      if(valid == need.size()){
        ans.add(left);
      }
      char d = s.charAt(left);
      if(need.containsKey(d)){
        if(windows.get(d).equals(need.get(d))){
          valid --;
        }
        windows.put(d, windows.get(d) - 1);
      }
      left ++;
    }
  }

  return ans;
}

# 581. 最短无序连续子数组 (opens new window)

思路：

  • 从左边到右边维护最大值及right（需要排序的连续元素的右指针），如果大于当前max，则说明正常，继续走，如果小于，则说明是被打乱的元素，更新right。
  • 同理更新min和left
public int findUnsortedSubarray(int[] nums) {

  int max = Integer.MIN_VALUE;
  int min = Integer.MAX_VALUE;
  int right = -1;
  int left = -1;
  int n = nums.length;

  for(int i = 0;i < n;i ++){

    if(nums[i] >= max){
      max = nums[i];
    }else {
      right = i;
    }

    if(nums[n - 1 - i] <= min){
      min = nums[n - 1 - i];
    }else {
      left = n - 1 - i;
    }
  }

  return right == -1 ? 0 : right - left + 1;
}

public int countSubstrings(String s) {

  int ans = 0;
  for(int i = 0;i < s.length();i ++){

    ans += countSubAns(s, i, i);
    if(i > 0){
      ans += countSubAns(s, i - 1, i);
    }
  }

  return ans;
}

private int countSubAns(String s, int left, int right){

  int cnt = 0;
  while(left >= 0 && right < s.length()){
    if(s.charAt(left) == s.charAt(right)){
      cnt ++;
      left --;
      right ++;
    }else {
      return cnt;
    }
  }

  return cnt;
}

对于计算“下一个变化值”的这种题目，一般用单调栈或单调队列

public int[] dailyTemperatures(int[] temperatures) {

  Deque<Integer> stk = new ArrayDeque<>();
  int n = temperatures.length;
  int[] ans = new int[n];

  for(int i = 0;i < n;i ++){
    while(!stk.isEmpty() && temperatures[stk.peek()] < temperatures[i]){
      int index= stk.pop();
      ans[index] = i - index;
    }
    stk.push(i);
  }

  return ans;
}
